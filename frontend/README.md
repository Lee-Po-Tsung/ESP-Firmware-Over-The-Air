# Frontend

## Getting Started

1. Go to the frontend directory:

```bash
cd frontend
```

2. Install dependencies:

```bash
npm install
```

3. Run in development mode (with hot reload):

```bash
npm run dev
```

## Build and Preview

1. Build the project:

```bash
npm run build
```

2. Preview the production build:

```bash
npm run preview
```
